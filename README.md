# Interview AI 🤖

An AI-powered interview preparation application that helps users practice for job interviews.

## Features
- Generate interview questions based on job role
- AI-powered response evaluation
- Practice and improve interview skills

## Tech Stack
- Python
- Flask
- AI/ML Integration

## How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python run.py`
